=================================================================
TROUBLESHOOTING
=================================================================

"Failed to connect to reCamera"
  → Check USB cable connection
  → Run: ping 192.168.42.1
  → Verify firewall allows RTSP (port 554)
  → Restart reCamera and reconnect

"Stream timeout triggered"
  → Already using TCP (configured for reliability)
  → Check network stability
  → Reduce FRAME_SKIP if too aggressive
  → Ensure reCamera is powered on

"Low FPS / Dropped frames"
  → Increase FRAME_SKIP (2 → 4 or 8)
  → Reduce INFERENCE_SIZE
  → Lower PERSON_CONFIDENCE threshold
  → Enable GPU (auto-detected)
  → Close other applications

"No detections"
  → Verify zone_config.py zone coordinates
  → Test with known fight video first
  → Check LSTM model path
  → Increase PERSON_CONFIDENCE if too low
